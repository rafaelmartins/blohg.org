blohg-0.5

blohg 0.5 is out!

This version comes with some improvements to the reStructuredText directives:

  • youtube directive now uses the Youtube iframe api, instead of the flask embed tag directly.
  • Added a new directive sourcecode, to do syntax highlight with Pygments.
  • Added a new directive subpages, to create a list of links for subpages automatically.

See the documentation of the reStructuredText directives for reference.

Thanks!

blohg-0.4

blohg 0.4 is out!

This version comes with attachments support and a few bug fixes:

  • enabled 404 error for static files. Thanks to Ry4an.
  • fixed module imports on the main script, that was importing system-wide modules when running from the source tarball.
