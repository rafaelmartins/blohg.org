blohg-0.7

blohg 0.7 was just released.

This version comes with some nice new features:

  • added label support for :attachment: roles.
  • added author name and email to the metadata API. Useful for show the author of each post in metadata.
  • the production setup now uses to the default branch instead of the tip revision. Useful for people who want to setup a branch for drafts. Just commit your drafts to a separated branch and merge them in default when done.

See the documentation for more info.

Thanks!

blohg-0.6

blohg 0.6 is out!

This version comes with support to Flask 0.7. blohg will no more work with older versions of Flask.

There's also some other minor improvements:

  • Fixed customized Jinja2 loader that gets templates from the Mercurial repository.
  • Improved handling of static files from the Mercurial repository.
  • Improved automatic reload of Mercurial stuff when a new changeset is pushed to the production server.

Thanks!
